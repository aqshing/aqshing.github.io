1.词汇

一、时间，空间

1. pre- 时间提前，预先 = before

  • prewar [ˈpriːˈwɔː]

    【记】pre-预先；war 战争→ 战前的

    【意】adj. 战前的

  • predict [prɪˈdɪkt]

    【记】pre- 预先；-dict- 说→ 预言

    【意】v. 预言，预测；预示，预告

    名词：prediction

    【意】n.预测；预言

    同义词：

    ① forecast

    【意】v/n. 预测、预料

    the weather forecast 天气预报

    ② foresee

    【意】v. 预见

词根拓展：

-dict-，-dic- = say 言，词，说

  • dictionary

    【意】n.字典；词典

    ion、ary为名词后缀

  • predict

    【意】v.预先；预告；预测

  • predictable

    【意】adj.可预测的

    able：可…的(可怎样的)

  • addict

    【意】v.使…沉迷；使…上瘾

    ad：去…；强调…

    sb be addicted to sb/sth

    使迷恋于…；使沉迷于…（后面只能跟坏的事情）

  • contradict

    【意】v.与…矛盾

    contra：反，逆

    名词：contradiction

    【意】n.字典；词典

    形容词：contradictory

    【意】adj.矛盾的；有争议的

  • indicate

    【意】v.暗示；指示；表明

    in：在里面；dic：说

  • precondition [priːkənˈdɪʃ(ə)n]

    【记】pre- 预先；condition 条件→ 前提条件

    【意】n. 前提条件，先决条件

  • preview [ˈpriːvjuː]

    【记】pre-预先；view 看→ 预览

    【意】v./n. 预览，预演，预先查看

词根拓展：

-vi-， - vis-,，- vid-， -view- = see 看

  • visible

    【意】adj.可见的

    ible：可能怎样的

  • invisible

    【意】adj.不可见的；看不见的

    in表否定

  • advice

    【意】n.建议

  • advise

    【意】v.建议；n.建议；通知

  • provide

    【意】n.规定；v.提供

    pro：空间向前，方向向前

    名词：provision

    【意】n.规定；条款；供应；供应品

  • revise

    【意】v.修改；修订

    re：再次，反复

    名词：revision

    【意】n.修改；修订

  • supervise

    【意】v.监督；监管

    super：超级，强调

    名词：supervision

    【意】n.监督；监管

    supervisor

    【意】n.监督者；导师

  • visual

    【意】adj.视觉的

    vis：看；ual：形容词后缀

  • viewpoint

    【意】n.观点；看法

  • interview

    【意】v/n.面试；访谈

  • review

    【意】n/v.复习；回顾

    re：反复，再一次

    review <=> go over 复习；回顾

  • evident

    【意】adj.明显的；显而易见的

    【e-；ex-】：向外，出去；ent：形容词后缀

    名词： evidence

    【意】n.明显；证据；迹象

2. pro- 空间向前，预先；方向向前(偶尔也表时间提前)

  • proceed [prəˈsi:d]

    【记】pro- 方向向前；-ceed-走 → 向前走，前进

    【意】 v. 进行，前进 [n. 收入，获利(一般用复数形式，即proceeds)]

  • proclaim [prəˈkleɪm]

    【记】pro- 方向向前；-claim- 叫、喊 → 向前喊出来，宣布

    【意】v. 宣告，公布，表明

    名词：proclamation

  • project [ˈprɒdʒekt]

    【记】pro- 方向向前；-ject- 投，射，扔 → 投射

    【记】pro- 时间提前；-ject- 投，射，扔 → 计划

    【意】v. 投影，投射，放映；计划；发射

    【意】n. 项目，工程；计划，规划

    Project Hope 希望工程

    projector 【意】n. 投影仪，放映机

    engineer 【意】n. 工程师

  • prospect [ˈprɒspekt]

    【记】pro- 方向向前；-spect- 看→往前看→前景，展望

    【意】n. 前景，展望，未来

词根拓展：

-spec-, -spect-, -spi-, -spic- = look 看

  • prospect

    pro：方向向前

    【意】n.前景；未来；展望

    形容词：prospective

    【意】adj.未来的；盼望中的

  • retrospect

    retro：(方向)向后看

    【意】v/n.回顾；追溯

  • inspect

    in：方向向里

    【意】v.调查；看法

    inspect <=> look into 调查

    inspection

  • aspect

    单独的a字母相当于e字母(即向外)

    【意】n.外貌；样子

  • expect

    ex：向外

    【意】v.期待；期盼

    名词：expectation

    【意】n.期待；期盼

    unexpected

    【意】adj.意料之外的；意想不到的

  • respect

    re：反复，一再

    【意】v/n.尊重；n.方面

    from all respects 从各方面而言

    形容词：respectable

    【意】adj.可敬的

    respective

    【意】adj.分别的；各自的

  • perspective

    per：渗透，一直

    【意】adj.透视的；n.透视图；观点

    in my perspective/from my perspective 在我看来

  • suspect

    su：方向向下

    【意】v.怀疑

    名词：suspicion

    【意】n.怀疑

    形容词：suspicious

    【意】adj.怀疑的

    be suspicious of… 怀疑…

  • conspicuous

    con：强调

    【意】adj.明显的；显而易见的

    conspicuous <=> evident

3. fore- 在…之前 = before

  • forearm [ˈfɔ:rɑ:m]

    【记】fore- 前；arm 手臂 → 前臂

    【意】n. 前臂； v. 预先武装，预先准备

    arm 【意】v. 武装

    army 【意】n. 军队

    sb be armed with sth 某人被…所武装

    eg：We usually be armed with knowledge.

    我们通常用知识武装自己。

  • forehead [ˈfɔ:hed]

    【记】fore- 前；head 头 → 前额

    【意】n. 额头；前部

  • forewarn [fɔ:ˈwɔ:n]

    【记】fore- 预先；warn 警告，告诫 → 预告

    【意】v. 警告，告诫；通知；预告

  • forecast [ˈfɔ:kɑ:st]

    【记】fore- 预先；-cast- 估算 → 预测

    【意】v. 预报，预测；预示 n. 预报；预言

    同义词：predict 预示，预告

    the weather forecast 天气预报

  • foresee [fɔ:ˈsi:]

    【记】fore- 预先；see 见 → 预见

    【意】v. 预知，预见

4. an-, anci-, ant-, ante- 在…之前

  • ancestor [ˈænsestə(r)]

    【记】an- 前；-cest-走；-or 表示人 → 先走的人 → 祖先

    【意】n. 祖先，祖宗

  • ancient [ˈeɪnʃənt]

    【记】anci- 之前；-ent 形容词后缀

    【意】adj. 古代的；古老的；过时的

  • antique [ænˈti:k]

    【记】ant(i) 前；-ique = -ic 形容词后缀

    【意】adj. 古老的，古代的；古董的

    【意】n. 古玩，古董

    antique shop 古玩店；古董店

  • antecedent [ˌæntɪˈsi:dnt]

    【记】ante- 前；-ced-走；-ent 为名词和形容词后缀

    【意】adj. 在前的，在先的，先行的 n. 祖先；先行词

词根拓展：

-ced-, -ceed-, -cess-, -cest- = go 走

  • precede

    【意】v. 领先，优先

  • exceed

    【意】v. 超出，超过，胜过

    数量上的程度和成都上的程度超出都可以

    名词：excess

    【意】n. 过多，过量，过分

    形容词：excessive

    ive：多数情况为形容词后缀

    【意】adj. 过多的，过量的，过分的

  • proceed

    【意】v. 前进，前行

  • process

    【意】n. 进程，过程

    【意】v.加工，处理

  • procedure

    ure：为名词后缀居多

    【意】n. 步骤，流程

  • succeed

    suc：同sub，表示向下

    【意】v. 继承；成功

    表达成功时：

    名词：success 【意】n.成功

    形容词：successful 【意】adj.成功的

    表达继承时：

    名词：succession【意】n.连续，继续

    successor 【意】n.继承人，接班人

    形容词：successive 【意】adj.连续的，继续的

  • recede

    re：表方向向后

    【意】v. 倒退，后退，退却

    名词：recession

    【意】n. 倒退，后退(还可以指经济衰退)

  • predecessor

    de：表示下；cess：走；or：表人

    【意】n. 前辈，先辈；祖先

  • antecedent

    ante：在…之前；ent：一般作形容词的后缀

    【意】adj. 先前的，在前的

    【意】n.前辈，祖先；(语法)先行词

  • ancestor

    【意】n. 祖先

  • necessary

    ne：相当于no，表否定；ary：形容词后缀

    【意】adj. 必要的，必须的

    动词：necessitate

    ate：动词后缀

    【意】v. 需要，必要

    名词：necessity

    ity：名词后缀

    【意】n. 必要性，必然性，必需品

  • access

    ac：表示去哪里；cess：为名词，表示走

    【意】n. 接口，入口；通道

    access to… 接近…，靠近…

    形容词：accessible

    ible：形容词后缀

    【意】：adj.可接近的，可靠近的

    assert 【意】v. 断言

5. post- 之后，后面

  • postgraduate [ˌpəʊstˈgrædʒuət]

    【记】post- 后；graduate 大学毕业生
    【意】adj. 研究生的；n. 研究生

    undergraduate 【意】n.在读大学生，大学肄业生

  • postwar [ˈpəʊstˈwɔ:]

    【记】post- 后；war 战争
    【意】adj. 战后的

    prewar 战前的

  • postpone [pəˈspəʊn]

    【记】post- 后；-pon- 放 → 放在原定时间之后 → 延期，推迟
    【意】v. 使延期，延缓

    postpone <=> delay 延期，延缓

词根拓展：

-pos-,-posi-, -pon- = put 放置

  • post

    【意】n. 邮件，职位

    【意】v. 发布

    repost 【意】v. 转发

    post office 邮局

  • position

    ion：名词后缀

    【意】n. 位置，职位，地位，境地

  • purpose

    pur：<=>pre，表时间提前

    【意】n. 目的，意图，打算

    purpose <=> aim，goal 目标，目的

    on purpose(当副词使用) 故意地

  • deposit

    de：方向向下；pos：表放置；it：走

    【意】v. 存款，储蓄；存放

    反义词：withdraw

    【意】v.提，取(钱)；撤销

    draw oneˈs attention 吸引某人的注意力

  • positive

    【意】adj. 积极的

    反义词：negative

    【意】adj. 负面的，消极的

  • compose

    com：表共同一起，来自come；pose：表放

    【意】v. 组成，构成

    名词：composition

    【意】n.组成，构成；构图，乐曲，乐章；作文

    component

    ent：作名词后缀，表达人或物

    【意】n.组成物，成分，部件

  • oppose

    op：表反或逆；pose：表放

    【意】v. 反对

    形容词：opposite

    ite：形容词后缀

    【意】adj. 反对的，对立的，相反的

    名词：opponent

    ent：作名词后缀，表人

    【意】n. 对手 adj.反对的

  • postpone

    【意】v.推迟，推延

  • expose

    [e- , ex-]：表方向向外

    【意】v. 暴露，揭露

    名词：exposure

    ure：为名词后缀

    【意】n.暴露，揭露

  • propose

    pro：方向向前；pose：放

    【意】v. 求婚；提议，建议

    名词：proposal

    al：名词后缀

    【意】n. 求婚；提议，建议

二、否定

1. ab-, abs- 相反的，反常的，否定

  • abnormal (=anormal) [æbˈnɔ:ml] ( [æˈnɔ:ml] )

    【记】ab- 相反；normal 正常的，正规的
    【意】adj. 反常的，异常的；不规则的

    normal 【意】adj. 正常的，正规的；师范的

    normal university 师范大学

  • abuse [əˈbju:s]

    【记】ab- 否定，反常；use 使用 → 不正常使用 → 滥用
    【意】n. 滥用；恶习；侮辱；恶言
    【意】v. 滥用；虐待；辱骂

    child abuse 虐待儿童

    authority abuse 权力滥用

  • absent [ˈæbsənt]

    【记】abs- 否定，离开；-ent 形容词后缀 → 离开的，缺席的
    【意】adj. 缺席的，不在场的；缺乏的；不在意

    名词：absence 【意】n. 缺席，缺失

    the absence of mind 心不在焉

2. anti- 对抗, 相反

  • antiwar [ˈæntɪˈwɔ:]

    【记】anti- 对抗；war 战争
    【意】adj. 反战的，反对战争的

  • anti-virus [ˈænti:vˈaɪərəs]

    【记】anti- 对抗；virus 病毒，毒素
    【意】adj.（软件）杀毒的，防病毒的

  • anti-dumping [ˈænti:dˈʌmpɪŋ]

    【记】anti- 对抗；dump 倾倒；-ing 形容词后缀
    【意】adj. 反倾销的

    antidumping law 反倾销法

  • antibody [ˈæntibɒdi]

    【记】anti- 对抗；body 身体
    【意】n. 抗体

3. dis- 不，没有

  • dishonest [dɪsˈɒnɪst]

    【记】[dis- 不；honest 诚实的]
    【意】adj. 不诚实的，不老实的；不正直的；狡猾的

    honesty 【意】n. 诚实，诚信

    dishonest <=> sly 【意】adj. 狡猾的

  • disobey [ˌdɪsəˈbeɪ]

    【记】[dis- 不；obey 服从，听从]
    【意】v. 不服从，不顺从

    obey 【意】v.服从，听从，遵守

    obedient 【意】adj.服从的，顺从的

    obedience 【意】n. 服从，听从

    同义词 syn.

    comply with 遵守，服从

    submit 【意】v.提交；顺从，服从

    submit to 提交给…；服从于… [动词和介词构成搭配]

    submissive 【意】adj.顺从的；服从的

  • disorder [dɪsˈɔ:də(r)]

    【记】[dis- 无；order 秩序]
    【意】n. 混乱，凌乱；动乱，骚乱；v. 使混乱，使凌乱；扰乱

    order 【意】n.顺序，秩序；命令，要求；订单 v.命令；下订单

  • discredit [dɪsˈkredɪt]

    【记】[dis- 无；credit 信誉，信用]
    【意】v. 使不可置信；使被怀疑； n. 丧失名誉

    credit card 信用卡

词根拓展：

-ord-, -ordin- = order 顺序

  • order

    【意】n. 顺序，秩序；命令，要求；订单 v.命令；下订单

    形容词：orderly

    【意】adj. 有序的，整齐的

    反义词：disorder

    【意】n. 混乱，凌乱；动乱，骚乱；v. 使混乱，使凌乱；扰乱

  • ordinary

    ordin：顺序；ary：形容词后缀

    【意】adj.普通的；平常的

    反义词：extraordinary

    【意】adj.不普通的；非凡的；极度的

  • coordinate

    co：合作，共同，一起；ordin：顺序；ate：动词后缀

    【意】v. 协调

词根拓展：

-cred- = believe 相信，信任

  • credit

    【意】n. 信誉，信用；学分

    反义词：discredit

    【意】v. 使不可置信；使被怀疑； n. 丧失名誉

    形容词：credible

    ible的形容词后缀表达:可…的

    【意】adj.可信的

    反义词：incredible

    in表否定

    【意】adj.令人难以置信的

    同义词 syn.

    unbelievable 【意】adj.难以置信的

    unimaginable 【意】adj.难以想象的

4. il-, im-, in-, ir- 不，无

if( 单词刚好以 l 字母开头 ) { 表否定意义时，添加 il- 为前缀表否定; }

else if( 以 m 开头 ) { 添加 im- 为前缀表否定; }

else if( 以 n 开头 ) { 添加 in- 为前缀表否定; }

else if( 以 r 开头 ) { 添加 ir- 为前缀表否定; }

else { 如果都不是，则添加 im- 或 in- 为前缀表否定; }

  • illegal [ɪˈli:gl]

    【记】il- 不；legal 合法的
    【意】adj. 不合法的，违法的；违反规则的
    n. 非法移民，非法劳工

  • illogical [ɪˈlɒdʒɪkl]

    【记】il- 不；logical 符合逻辑的，合乎常理的
    【意】adj. 不合逻辑的；无意义的

    logical 【意】adj.逻辑的

    logic 【意】n.逻辑

    logistics 【意】n.物流学；后勤学；运筹学

    philosophy 【意】n.哲学

    psychology 【意】n.心理学

    psychiatry 【意】n.神经学

  • immature [ˌɪməˈtjʊə(r)]

    【记】im- 不；mature 成熟的
    【意】adj. 不成熟的；未完成的；粗糙的

    ripe 【意】adj.成熟的

    maturity 【意】n.成熟
